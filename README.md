# BookSpace

## Introduction
A simple way to organize your reading.


## Tech Stack
- Golang - backend language
- ReactJS - UI technology
- Couchbase - DB to store tasks
- Digital Ocean - hosting for the app

## Steps to run

1. Clone the repo
2. Run the server
    - go run [path to main]
3. Run the client - open new terminal
    - cd ui
    - npm start
